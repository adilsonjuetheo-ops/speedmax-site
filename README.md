# Speed Max Internet

Landing page criada do zero para a Speed Max Internet, com foco em conversao para WhatsApp e identidade visual inspirada na paleta do Instagram e do banner enviado.

## Arquivos

- `index.html`: estrutura principal da pagina
- `styles.css`: visual, responsividade e animacoes
- `script.js`: menu mobile e revelacao no scroll
- `assets/logo-speedmax.svg`: logo redesenhada em SVG
- `assets/hero-rocket.svg`: ilustracao principal do hero

## Como abrir

Abra `index.html` diretamente no navegador ou rode um servidor local simples:

```powershell
python -m http.server 4173
```

Depois acesse `http://127.0.0.1:4173/`.

## Observacao

O botao `Central do Assinante` foi configurado provisoriamente para abrir o WhatsApp com mensagem pronta, porque o link real da central nao estava legivel no material enviado.
